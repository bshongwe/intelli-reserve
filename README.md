# IntelliReserve – Real-Time Booking Engine + Escrow Intelligence Suite

A fintech-grade booking platform with dynamic pricing, escrow payments, and analytics.

## Quick Start

```bash
make up
# Then start individual services from their directories
```

## Project Structure
- `frontend/` – React/Next.js UI
- `backend/` – Go microservices (core logic)
- `bff/` – Backend-for-Frontend (TypeScript)
- `infra/` – Terraform + Kubernetes
